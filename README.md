# 🏡 EliteState

A modern **Real Estate Flutter Application** built using **Flutter**, **Firebase**, **Provider**, and **MVVM Architecture**. EliteState enables users to sign up, log in, browse property listings, and add their own properties through a clean and responsive user interface.

---

## ✨ Features

- 🔐 Firebase Authentication
- 👤 User Registration & Login
- 🏠 Add New Property
- 📋 Browse Property Listings
- 📍 Property Details
- 👤 User Profile
- ☁️ Cloud Firestore Database
- 📱 Responsive UI
- 🎨 Dark Theme
- ⚡ Provider State Management
- 🏗 MVVM Architecture

## 🚀 Getting Started

### Clone Repository

```bash
git clone https://github.com/your-username/EliteState.git
```

### Go to Project

```bash
cd EliteState
```

### Install Dependencies

```bash
flutter pub get
```

### Configure Firebase

1. Create a Firebase project.
2. Enable **Firebase Authentication**.
3. Enable **Cloud Firestore**.
4. Add the Firebase configuration files:
   - `google-services.json` (Android)
   - `GoogleService-Info.plist` (iOS)

Run:

```bash
flutterfire configure
```

### Run the Application

```bash
flutter run
```
